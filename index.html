// 01 sobre-mim

Quem é Diego

Sou desenvolvedor front-end em formação contínua, apaixonado por transformar layouts e ideias em interfaces reais, funcionais e agradáveis de usar. Gosto de entender o "porquê" por trás de cada decisão de UI — não só encaixar componentes, mas construir experiências coerentes.

Meu objetivo é conquistar uma oportunidade de estágio ou vaga júnior onde eu possa aprender com times experientes, contribuir com código limpo e evoluir para construir produtos com impacto real para os usuários.

  • Atenção aos detalhes — pixels, espaçamentos e estados importam.
  • Curiosidade técnica — gosto de entender como as coisas funcionam por dentro.
  • Colaboração — código é feito para ser lido por outras pessoas também.
  • Consistência — pratico todos os dias, mesmo que pouco.
